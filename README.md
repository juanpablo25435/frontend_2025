Instalar Angular
````
npm i -g @angular/cli@14.2.12 --force
````
Clonar el repositorio
````
git clone https://github.com/juanpablo25435/frontend_2025.git
````
Acceder a la carpeta "frontend"
````
cd frontend_2025
````
Instalar las dependencias
````
npm install --force
````
Correr el servidor
```
ng serve
```
